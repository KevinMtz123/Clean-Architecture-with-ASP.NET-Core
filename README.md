# DientesLimpios 🦷

Sistema de gestión para clínica dental desarrollado con **.NET** siguiendo los principios de **Clean Architecture**.

## 🏗️ Arquitectura

El proyecto está estructurado en capas bien definidas:

- **API** - Capa de presentación (ASP.NET Core Web API)
- **Aplicacion** - Casos de uso y lógica de negocio
- **Dominio** - Entidades y reglas de negocio core
- **Identidad** - Autenticación y autorización
- **Infraestructura** - Servicios externos e implementaciones
- **Persistencia** - Acceso a datos y repositorios
- **Pruebas** - Tests del sistema

## 🛠️ Tecnologías

- ASP.NET Core
- Entity Framework Core
- Clean Architecture
